# ednl-graph-java
Graph implemented in java, using matrix, where each cell has a linked list.

Compile for bin folder
```bash
javac -Xlint -d bin src/graph/*.java
```

Run from bin folder
```bash
java -classpath bin graph.Main
```

Application content for academic purposes
